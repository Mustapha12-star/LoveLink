
# LoveLink

LoveLink هو تطبيق تعارف يساعد الأشخاص من مختلف أنحاء العالم على التعرف وبناء علاقات حقيقية.
LoveLink is a dating app that helps people from around the world meet and build real relationships.

## Features
- إنشاء ملف شخصي (Create a profile)
- البحث عن أصدقاء جدد (Find new friends)
- الدردشة الفورية (Instant chat)
